⚙ Lokaler Server starten

Der lokale Modus braucht streetview_server.py als Bruecke zwischen Browser und Python. Starte ihn einmalig in einem Terminal: 

cd streetview-explorer
python3 streetview_server.py

Der Server startet auf http://localhost:5000 und installiert fehlende Pakete (flask, flask-cors) automatisch.

⚙ Konfiguration

🔎 Google Maps API-Key erstellen

  1. Oeffne console.cloud.google.com
  2. Neues Projekt erstellen (oder bestehendes auswaehlen)
  3. APIs & Dienste → Bibliothek → „Map Tiles API“ → Aktivieren
  4. APIs & Dienste → Anmeldedaten → + Anmeldedaten erstellen → API-Schluessel
  5. Abrechnung aktivieren (Pflicht; kostenlos bis 100.000 Tiles/Monat)
  6. Key kopieren → hier einfuegen

Kosten: low ~3.125 · medium ~781 · high ~195 Panoramen/Monat im Freikontigent.

🔑 GitHub PAT-Token

Nur im RAM — nie gespeichert. Wie erstelle ich einen Token? ↓

📖 PAT-Token erstellen (2 Min)

  1. Oeffne github.com/settings/tokens → Fine-grained token
  2. Name: streetview-explorer
  3. Expiration: 30 days
  4. Repository access: Only: streetview-explorer
  5. Permissions: Actions → Read and write (nur das!)
  6. Generate → kopieren → hier einfuegen
