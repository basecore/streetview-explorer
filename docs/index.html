🔑 GitHub Personal Access Token (PAT)

Token wird nur im RAM gehalten — nie gespeichert oder uebertragen (ausser an api.github.com). Wie bekomme ich einen Token? ↓

PAT-Token erstellen (2 Minuten):
  1. Oeffne github.com/settings/tokens
  2. Klicke Generate new token → Fine-grained token
  3. Name: z.B. streetview-explorer
  4. Expiration: z.B. 30 days
  5. Repository access: Only select repositories → streetview-explorer auswaehlen
  6. Permissions → Actions: Read and write
  7. Klicke Generate token → Token kopieren → hier einfuegen

Sicherheit: Nur Actions: Read & Write berechtigung — kein Zugriff auf Code, Secrets oder andere Repos.
