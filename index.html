🔑 Google Maps API-Key (Map Tiles API)

Nicht getestet
📖 Wie bekomme ich einen Google API-Key? (aufklappen)
  1. Oeffne console.cloud.google.com
  2. Neues Projekt erstellen oder bestehendes auswaehlen
  3. APIs & Dienste → Bibliothek → "Map Tiles API" → Aktivieren
  4. APIs & Dienste → Anmeldedaten → + API-Schluessel erstellen
  5. Abrechnung aktivieren (Pflicht, auch fuer kostenlosen Tier)
  6. Key kopieren → oben einfuegen → testen

Kosten: Kostenlos bis 100.000 Tiles/Monat. low=32 · medium=128 · high=512 Tiles/Panorama.

Nur Map Tiles API benoetigt — keine Geocoding API erforderlich.

Tool basiert auf streetview-dl von Matt Stiles.

🔑 GitHub PAT-Token (nur Cloud-Modus)

Nicht getestet
📖 Wie erstelle ich einen PAT-Token? (aufklappen)
  1. Oeffne github.com/settings/tokens → Fine-grained token
  2. Name: streetview-explorer   Expiration: 30 days
  3. Repository access: Only select repositories → streetview-explorer
  4. Permissions: Actions → Read and write
  5. Generate → kopieren → oben einfuegen → testen

Token wird nur im Browser-RAM gehalten — nie gespeichert.
